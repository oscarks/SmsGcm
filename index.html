SMS Google Cloud Message


The SMS GCM Project  permit  to send  SMS from a grails web app  using a android device with SMS facility. It is composed of two modules, a server (web) where the messages are generated and a android aplication client, responsible for sending the SMS.  The applications use the Google Cloud Message for Android to communicate with each other.


Modules

- SmsGcmServer plugin - a grails plugin, implements the server side. The mechanism for sending SMS is implemented in a service and also provides a basic interface for sending messages.
- SmsGcmAndroClient - a android application.

Steps to use

1 - Create a Google API Project and enable the GCM service

To use the SMS GCM you need first create a Google API project. Visit the GCM site to know how do it (Getting Started). There you will also find the information on what is the GCM is and how it works. Is very important to create a server API Key, you will need it to configure the server. Anotate the project id, you will use it to configure the client side.

2 - Server Configuration

The server is a grails web application. First all you need to create a grails application and install the SmsGcmServer plugin. See Grails Quick Start guide to know how to create a grails project.
To plugin install you have several options. For more information go to Grails documentation site.
To configure the server is necessary to setup the key api in the config.groovy file. Open the config.groovy file (in conf directory) and add the line below:

google.api.key='XXXXXXXXXXXXXXXX'

where XXXXXXXXXXXXXXXX is the key generated in the step 1.
